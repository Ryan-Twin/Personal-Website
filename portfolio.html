Implementation Focus

Painteractive

An assignment for my Human-Computer Interaction Capstone course, this project is a collaboration with several other students, where the design and implementation, while constrained due to the limitations of the course, largely reflect my contribution. The front-end was implemented using React, while the back-end utilizes Express, Node.js, and MongoDB. The project was hosted using AWS EC2.

The project is still online here and source code is also available upon request.

TooManyGames Database

For my Intro to Database Management Systems course, I was tasked in a small group with creating a web app representation of a video game library, featuring thousands of games from Steam and creating a portal where users can interact with each other, mark which games they owned, were interested in, and more. Much like Painteractive, The front end was also implemented with React, while the back end features a SQL Relational Database instead of MongoDB, and is similarly hosted on AWS EC2.

The project is still online here and source code is also available upon request.
